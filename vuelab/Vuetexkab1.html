Vue.js + MathJax + 程式碼高亮展示

Vue 程式碼範例：


<div id="app">
  <p>{{ message }}</p>
  <button @click="reverse">Reverse</button>
</div>

<script>
const { createApp } = Vue;
createApp({
  data() {
    return { message: 'Hello Vue!' };
  },
  methods: {
    reverse() {
      this.message = this.message.split('').reverse().join('');
    }
  }
}).mount('#app');
</script>
