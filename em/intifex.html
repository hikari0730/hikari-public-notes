Derivation of the Integrating Factor Method

This note presents the derivation of the Integrating Factor Method used to solve first-order linear ordinary differential equations (ODEs).

1. General Form

The standard form of a first-order linear ODE is:

$$\frac{dy}{dx} + P(x)y = Q(x)$$

Our objective is to find the general solution for \(y(x)\).

2. Introducing the Integrating Factor

We define the integrating factor \(\mu(x)\) as:

$$\mu(x) = e^{\int P(x) \, dx}$$

Multiplying both sides of the differential equation by \(\mu(x)\):

$$\mu(x)\frac{dy}{dx} + \mu(x)P(x)y = \mu(x)Q(x)$$

The left-hand side simplifies to:

$$\frac{d}{dx}[\mu(x)y] = \mu(x)Q(x)$$

3. Solving by Integration

Integrate both sides with respect to \(x\):

$$\mu(x)y = \int \mu(x)Q(x)\,dx + C$$

Solve for \(y(x)\):

$$y(x) = \frac{1}{\mu(x)}\left(\int \mu(x)Q(x)\,dx + C\right)$$

4. Summary

The general solution to:

$$\frac{dy}{dx} + P(x)y = Q(x)$$

is:

$$y(x) = \frac{1}{e^{\int P(x)\,dx}}\left(\int e^{\int P(x)\,dx} Q(x)\,dx + C\right)$$

5. Example

Solve the differential equation:

$$\frac{dy}{dx} + 2y = e^{-2x}$$

Step 1: Identify \(P(x) = 2\), \(Q(x) = e^{-2x}\)
Step 2: Compute the integrating factor:
$$\mu(x) = e^{\int 2 dx} = e^{2x}$$
Step 3: Multiply both sides:
$$e^{2x} \frac{dy}{dx} + 2e^{2x}y = 1$$
Left-hand side becomes:
$$\frac{d}{dx}[e^{2x}y] = 1$$
Step 4: Integrate both sides:
$$e^{2x}y = \int 1\,dx = x + C$$
Step 5: Solve for \(y\):
$$y = e^{-2x}(x + C)$$

Final Answer:

$$y(x) = e^{-2x}(x + C)$$
